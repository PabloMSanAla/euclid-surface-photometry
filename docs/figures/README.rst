Figures directory
==================

Place the paper figures here as PDF or PNG files.  The documentation
references the following figure names (taken from the paper
``Euclid_disc_breaks.pdf``):

Required figures
-----------------

- ``Euclid_Pipeline.pdf``   — Full pipeline illustration (Fig. 2 in the paper).
  Showcases all five modules for galaxy EUCL J040743.76−465230.1.

- ``EUC_SEGMID_-612927984481008950_mask_segment.pdf``   — Mask comparison
  figure for EUCL J040510.27−444836.3 (Fig. 3 in the paper).

- ``sample_criteria.pdf``   — Stellar mass vs. redshift for the selected sample
  (Fig. 1 in the paper).

- ``average_profiles_VIS.pdf``   — Stacked normalised SB profiles by break type
  (Fig. 5 in the paper).

- ``breaks_histograms_separated.pdf``   — Distributions of structural parameters
  by break type (Fig. 6 in the paper).

- ``rbreak_vs_h.pdf``   — Break radius vs. inner / outer scale length
  (Fig. 7 in the paper).

- ``Type_vs_redshift_grid.pdf``   — Evolution of disc break fractions with
  redshift (Fig. 8 in the paper).

All figures are from Sánchez-Alarcón et al. (2026, A&A).
The paper PDF ``Euclid_disc_breaks.pdf`` is included here for reference.
